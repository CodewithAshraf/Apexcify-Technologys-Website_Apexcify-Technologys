# Apexcify Technologies | Corporate Web Suite

A modern, high-performance multi-page website built for **Apexcify Technologies**. This project features a cinematic video-background landing page, an interactive project proposal system, and a unified design framework.

## 🚀 Live Demo
https://apexcify-technologyss.netlify.app/

## 🛠️ Tech Stack
* **HTML5** - Semantic structure for SEO optimization.
* **CSS3** - Custom properties (variables), Flexbox, and Grid for a responsive, modern UI.
* **JavaScript (Vanilla)** - Intersection Observer API for scroll animations and form handling.
* **Google Fonts** - 'Inter' typeface for a clean, tech-focused aesthetic.

## 📂 Project Structure
This project follows a "Single-Source" styling architecture where multiple HTML files rely on a centralized CSS and JS file for maximum maintainability.

```text
├── index.html          # Landing Page (Video Hero, Services, Process)
├── about.html          # Company Mission & Values
├── contact.html        # Direct Inquiry Page with Map
├── get-started.html    # Specialized Project Proposal Form
├── privacy.html        # Legal / Privacy Policy
├── security.html       # Security Protocols & Compliance
├── style.css           # Global Styles & Brand Variables
└── script.js           # Global Interactive Logic & Animations
